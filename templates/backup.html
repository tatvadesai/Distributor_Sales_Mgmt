To enable database backups, you need to configure Supabase credentials:

  1. Create a Supabase account at https://supabase.com
  2. Create a new project
  3. Create a table named backups with the following columns:
    • id (type: uuid, primary key)
    • backup_id (type: text)
    • table_name (type: text)
    • timestamp (type: text)
    • data (type: text)
  4. Get your Supabase URL and API key from the project settings
  5. Set the following environment variables:
    • SUPABASE_URL: Your Supabase project URL
    • SUPABASE_KEY: Your Supabase API key
  6. Restart the application
